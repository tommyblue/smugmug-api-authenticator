SmugMug API Authenticator

This app will help you get OAuth1 credentials from SmugMug. Credentials can be used to configure the SmugMug Backup App. This web app is open-source and can be found on GitHub. You can use it from this page or you can clone the repository and run your own instance. If you prefer other ways to get your credentials, check the SmugMug Backup App Readme file

Get your credentials

Insert key and secret for your app. If you don't have one, get them here. You'll be first redirected to the SmugMug website to enter your credentials and then back here to copy the token and secret keys you'll need to paste into the SmugMug backup app.
